\# Finance Analysis Project



This project explores financial investment behaviour using Python and Jupyter Notebook.



\## Key Features

\- Data cleaning and preprocessing

\- Exploratory data analysis (EDA)

\- Visualisations using Matplotlib and Seaborn



\## Tools Used

\- Python

\- Pandas

\- Matplotlib

\- Seaborn

\- Jupyter Notebook



\## Project Structure

\- data/: raw dataset

\- notebooks/: analysis notebook
